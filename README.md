# Smart Resume For Internship Web App Plan

## 项目定位

Smart Resume For Internship Web App Plan 是一个面向实习投递场景的智能简历网页应用规划与原型项目。

核心目标：

1. 维护一份完整的通用简历经历库
2. 输入目标岗位 JD
3. 根据 JD 关键词匹配相关经历
4. 生成特定岗位简历
5. 支持手动微调
6. 一键导出 PDF

## 迭代保存规则

每次迭代都在本母目录下新建一个时间戳文件夹：

```text
yyyyMMddHHmm
```

每个迭代文件夹应包含：

```text
Smart Resume For Internship Web App Plan/
  202605200927/
    index.html
    source/
      app.js
      styles.css
    docs/
      PRD.md
      CHANGELOG.md
```

## 文件说明

- `index.html`：当前迭代可直接访问的网页入口
- `source/`：源代码，包含样式、脚本、数据结构和核心交互逻辑
- `docs/PRD.md`：设计框架、产品需求、流程和数据结构说明
- `docs/CHANGELOG.md`：本次迭代更新日志
- `public/models/`：3D 模型等静态资源
